# Сложно сосредоточиться

## Описание проекта

«Сложно сосредоточиться» — это адаптивный лендинг, демонстрирующий современные подходы к вёрстке с приоритетом доступности и пользовательского комфорта. Проект представляет собой информационный сайт о методах концентрации внимания в эпоху информационного шума.

**Ключевые особенности:**

- Полностью адаптивная вёрстка с подходом Desktop First (3 ключевых разрешения: мобильные устройства, планшеты, десктопы)
- Поддержка светлой и тёмной темы с автоматическим переключением через prefers-color-scheme
- Ручное переключение темы через кнопки «День», «Неон», «Авто» с сохранением выбранного состояния
- Семантическая вёрстка с учётом стандартов доступности (a11y)
- Ленивая загрузка изображений галереи (loading="lazy")
- Декоративные элементы (уголки, плашка REC), реализованные через CSS-псевдоэлементы без использования дополнительных изображений

**Стек технологий:**

- HTML5 — семантическая разметка
- CSS3 — кастомные свойства (CSS-переменные), гриды, флексы, медиазапросы
- Адаптивная типографика — функция clamp() для резиновых шрифтов
- Методология БЭМ — для именования классов (naming convention)
- JavaScript — переключение тем (предоставлен в стартовом шаблоне)

## Выводы по результатам проекта

В ходе выполнения проектной работы были достигнуты следующие результаты:

1. **Реализована полноценная поддержка двух цветовых тем** через систему CSS-переменных. Переопределение всего 8 переменных в файлах dark.css и light.css позволяет управлять цветовой схемой всего сайта, что делает процесс ребрендинга или добавления новых тем максимально простым.

2. **Обеспечена бесшовная адаптивность** для всех типов устройств. Благодаря подходу Mobile First и грамотно настроенным медиазапросам, сайт корректно отображается на экранах от 320px до 1920px и выше, без горизонтального скролла и «разрывов» вёрстки.

3. **Повышена доступность сайта** для пользователей с ограниченными возможностями. Использование семантических тегов, атрибутов alt для всех изображений, скрытие декоративного контента от скринридеров (aria-hidden) и правильная стилизация фокуса (:focus-visible) позволяют пользователям с клавиатурной навигацией и программами чтения с экрана полноценно взаимодействовать с сайтом.

4. **Достигнута высокая производительность**. Ленивая загрузка изображений в галерее сокращает начальный вес страницы и ускоряет отображение первого экрана.

5. **Создана система переключения тем с визуальной обратной связью**. Кнопки выбора темы имеют все необходимые состояния (активная, hover, focus), что делает интерфейс интуитивно понятным и «живым».

6. **Реализованы сложные декоративные элементы** (уголки в шапке и подвале, плашка REC с точкой) без использования лишних HTML-элементов и изображений — только CSS-псевдоэлементы и абсолютное позиционирование.
